# Quote

The project is a quote application where you can share your favorite quote

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

What things you need to install the software and how to install them

```
you need to have browser
```


## Usage

* You input your name ,the name of the author and the quote

* you click add a quote

* you can upvote or downvote a quote by clicking on buttons

* you can also see details of the quote by clicking the show details button

* you can also delete a quote by clicking on delete button
